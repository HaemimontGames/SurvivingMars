[*Back to Index*](index.md.html) Tech Field =============== This mod item allows you to define a new technology field. Information about existing technologies can be found [here](LuaFunctionDoc_TechnologyNames.md.html). Properties ---------- Name (self.display_name) : Display name of the Tech Field, visible in the Research dialog. Description (self.description) : Tech Field description text. Research Costs (self.costs) : Required Research Points For Each Tech Slot. Discoverable (self.discoverable) : The tech in this field can be discovered through normal means. Show in Other Field (self.show_in_field) : The technologies in this field are visible in another UI group. Globals ---------- TechFields : List of all Tech Fields by name. (insert footer.md.html here)
